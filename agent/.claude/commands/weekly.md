---
description: 创建周刊
---

请你作为一名资深的技术编辑，帮助我创建一份关于人工智能生成内容（AIGC）的周刊。

要求：

1. 使用用户输入的时间($1)获取周数，如果用户没有提供时间，则自动计算当前时间所在的周数，整个任务都应该遵守这个时间。
2. 自然周使用“周日到周六”计算，推荐使用 `date -d "日期字符串" +%U` 方式计算周数。
3. 周刊标题格式：“Agili 的 AIGC 周刊（Y25W10）”, “Y25W10” 代表周刊的年份和周数。
4. 周刊以 “aigc-weekly-y25-w10.md” 的名称格式存储在当前目录，内容是最终可发布的周刊内容。

请按照以下步骤操作，协调各个 Agent 完成工作：

1. **收集内容 (Phase 1)**：
   - 调用 `researcher` Agent。
   - 任务：根据计算出的时间范围，抓取最新资讯。
   - 产出：`drafts` 目录下的原始内容。

2. **筛选信息 (Phase 2)**：
   - 调用 `editor` Agent。
   - 任务：对 `drafts` 中的内容进行筛选、去重和打分。
   - 产出：`drafts.yaml` 高价值内容列表。

3. **撰写内容 (Phase 3)**：
   - 调用 `writer` Agent。
   - 任务：基于 `drafts.yaml` 撰写最终周刊。
   - 产出：最终的 Markdown 周刊文件。

4. **审核与修订 (Phase 4)**：
   - 调用 `reviewer` Agent。
   - 任务：审核上一步生成的周刊文件。
   - **循环逻辑**：
     - 如果 Reviewer 返回 "PASS"，则任务完成。
     - 如果 Reviewer 返回修改意见，调用 `writer` Agent 根据意见进行修改。
     - 再次调用 `reviewer` 进行复核。
     - 此过程最多重复 3 次。如果 3 次后仍未通过，请保存当前版本并提示用户人工介入。

请务必一次性完成所有任务，过程中无需向我确认，向我呈现最终的周刊内容。
